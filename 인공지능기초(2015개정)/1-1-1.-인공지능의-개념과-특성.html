인공지능(AI)이란 무엇일까요?

선생님이 여러분에게 "오늘 기분이 어때?"라고 물어보면, 여러분은 표정이나 목소리를 통해 대답을 하죠? 이처럼 스스로 생각하고, 상황을 파악하고, 판단하는 능력을 우리는 '인간 지능'이라고 부릅니다.

인공지능(Artificial Intelligence)은 바로 이 인간의 지능을 기계나 컴퓨터 프로그램으로 똑같이 만들어보려고(구현하려고) 한 것입니다.

잠깐! 흔히 하는 오해 (오개념 방지)

"스마트폰에 있는 앱은 다 인공지능 아닌가요?"
아닙니다! 우리가 쓰는 많은 프로그램은 개발자가 미리 정해놓은 규칙대로만 움직이는 '일반 소프트웨어'입니다. 인공지능은 단순히 규칙을 따르는 것을 넘어, 스스로 상황을 판단하고 유연하게 대처하는 능력이 있어야 합니다.

인공지능 vs 일반 소프트웨어

그렇다면 인공지능과 일반 소프트웨어는 구체적으로 어떻게 다를까요? 단계별로 차이점을 알아봅시다.

일반 소프트웨어
  • 정해진 규칙대로만 똑같이 작동합니다.
  • 예상치 못한 입력이 들어오면 에러가 나거나 멈춥니다.
  • 예시: 음악 재생 프로그램
    사용자가 '재생' 버튼을 눌러야만 음악이 나옵니다.
인공지능 (AI)
  • 사용자의 요구나 주변 상황에 맞게 적응합니다.
  • 입력이 조금 달라도 유연하게 처리하고 추론합니다.
  • 예시: 음악 추천 소프트웨어
    내가 평소 듣는 취향과 현재 날씨를 분석해 알아서 음악을 골라줍니다.

✨ 핵심 쏙쏙 정리

인공지능을 사용하면 기계가 스스로 판단하고 도와주기 때문에, 사람이 일일이 신경 써야 하는 인지적 노력(생각하는 힘)이 훨씬 적게 든다는 큰 장점이 있습니다.

자율주행 자동차
software_vs_ai_logic.c 

// 1. 일반 소프트웨어의 논리: 정해진 규칙(if~else)에 의존합니다.
if (button_pressed == TRUE) {
    play_music("song_A.mp3"); // 버튼을 누르면 무조건 이 노래만 틉니다.
} else {
    stop_music();
}

// ---------------------------------------------------------

// 2. 인공지능의 논리: 상황을 분석하고 스스로 판단(예측)합니다.
String user_mood = analyze_face_expression(camera_input); // 표정을 분석해 기분 파악
String recommended_song = ai_model.predict(user_mood);    // 기분에 맞는 노래 추론/예측

play_music(recommended_song); // 알아서 어울리는 노래를 틀어줍니다.

* 위 코드는 이해를 돕기 위해 작성된 간단한 가짜 코드(의사코드)입니다. 일반 소프트웨어는 조건문(if)에 갇혀있지만, 인공지능은 상황을 분석(analyze)하고 예측(predict)하는 기능이 핵심입니다.
